flex的基本介绍

flex 即 flexible box 是弹性布局, 任何一个容器都可以指定为 flex 布局, 行内元素也可以设置, 设置 flex 布局后, 子元素的 float、clear、vertical-align 属性无效

设置 flex 后, 这些元素称为 flex容器, 而它们的所有子元素自动成为容器成员, 称为 flex item 

			.box {
				display: -webkit-flex; /* 兼容 webkit 内核 */
				display: flex;
			}
		
			.inline-box {
				display: inline-flex; /* 行内元素 flex*/
			}

flex的属性

容器上的6个属性

  • flex-direction 主轴的排列方向
  • flex-warp
  • flex-flow
  • justify-content
  • align-items
  • align-content

1、flex-direction属性

flex-direction 有4个值, 决定 主轴水平方向和排列起点 或 垂直方向和排列起点

.box { flex-direction: row || row-reverse || column || column-reverse; }

2、flex-warp

flex-warp 有3个值, 定义元素如果在一条轴线下排不下如何换行

.box { flex-warp: nowrap || wrap || warp-reverse; }

3、flex-flow

flex-flow 为 flex-direction 和 flex-warp 的简写形式

.box { flex-flow: [flex-direction] || [flex-warp]; }

4、justify-content

flex-flow 有5个值, 决定元素主轴方向的对齐方式, 它们具体的对齐方式与 主轴的水平方向有关系

.box { justify-content: flex-start || flex-end || center || sapce-between || space-around ; }

5、align-items

align-items 有5个值, 决定元素交叉轴方向的对齐方式

.box { align-items: stretch || flex-start || flex-end || center || baseline ; }

1 2 3 4

6、align-content

flex-flow 有6个值, 定义多条轴线的对齐方式, 如果只有一条轴线该属性不起作用

.box { justify-content: stretch || flex-start || flex-end || center || sapce-between || space-around ; }
